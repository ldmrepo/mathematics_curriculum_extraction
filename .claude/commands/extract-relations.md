# 성취기준 관계 추출

특정 성취기준의 모든 관계를 분석하고 추출합니다.

분석 내용:
1. **선수학습 관계**
   - 이 성취기준을 배우기 전 필요한 내용
   - 학년/영역 정보 포함

2. **후속학습 관계**
   - 이 성취기준 이후 학습할 내용
   - 확장/심화 개념

3. **유사 개념**
   - 같은 영역 내 유사 성취기준
   - 다른 영역의 관련 개념

4. **교육적 맥락**
   - 핵심 아이디어와의 연결
   - 실생활 적용 예시

성취기준 코드를 입력하거나 다음과 같이 요청하세요:
- "[2수01-01]의 관계 분석"
- "초등 3-4학년 분수 관련 성취기준 관계"
