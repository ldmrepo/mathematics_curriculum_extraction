## 1) 스키마(전체 구조) 다이어그램

```mermaid
graph TD
  %% ===== LAYER 1: 편제/과목 =====
  subgraph L1[편제/과목]
    SUB[Subject: 수학]
    PROG[Program: 공통/선택]
    COURSE[Course: 공통수학1/기본수학1/대수...]
  end

  %% ===== LAYER 2: 영역/내용요소 =====
  subgraph L2[영역·내용요소]
    DOM1[Domain: 수와 연산]
    DOM2[Domain: 변화와 관계]
    DOM3[Domain: 도형과 측정]
    DOM4[Domain: 자료와 가능성]
    CE[ContentElement: 내용요소 예: 겉넓이/부피, 다항식 연산...]
  end

  %% ===== LAYER 3: 성취기준/성취수준 =====
  subgraph L3[성취기준·성취수준]
    STD[AchievementStandard: 성취기준 예: 9수03-08]
    A[A수준]
    B[B수준]
    C[C수준]
    D[D수준]
    E[E수준]
  end

  %% ===== LAYER 4: 평가/지침/용어 =====
  subgraph L4[평가·지침·용어]
    ASMT[AssessmentItem: 예시 평가도구]
    GL[Guideline: 지도·평가 유의]
    LEX[LexiconTerm: 동의어/표현 사전]
  end

  %% ---- 구조적 관계 (문서 추출 기반) ----
  PROG -->|BELONGS_TO| SUB
  COURSE -->|UNDER_PROGRAM| PROG
  COURSE -->|HAS_DOMAIN| DOM1
  COURSE -->|HAS_DOMAIN| DOM2
  COURSE -->|HAS_DOMAIN| DOM3
  COURSE -->|HAS_DOMAIN| DOM4
  CE -->|IN_DOMAIN| DOM3
  STD -->|IN_DOMAIN| DOM3
  STD -->|SPECIFIED_BY| A
  STD -->|SPECIFIED_BY| B
  STD -->|SPECIFIED_BY| C
  STD -->|SPECIFIED_BY| D
  STD -->|SPECIFIED_BY| E
  ASMT -->|FOR_STANDARD| STD

  %% ---- 추론/고급 관계 (룰/모델 기반) ----
  CE -. ALIGNS_WITH .-> STD
  STD -. PREREQ .-> STD
  E -->|PROGRESSES_TO| D -->|PROGRESSES_TO| C -->|PROGRESSES_TO| B -->|PROGRESSES_TO| A
  DOM3 -. GUIDED_BY .-> GL
  CE -. GUIDED_BY .-> GL
  STD -. GUIDED_BY .-> GL
  CE -. HAS_LEXICON .-> LEX
  STD -. SIMILAR_TO .- STD
  STD -. EQUIVALENT_TO .- STD
```

## 2) 예시 인스턴스 다이어그램(도형·겉넓이/부피)

```mermaid
graph TD
  %% 노드
  SUB[수학]
  PROG[선택 중심]
  COURSE[공통수학1]
  DOM3[도형과 측정 (03)]
  CE1[내용요소: 겉넓이/부피]
  STD1[성취기준: 9수03-08<br/>"기둥·뿔·구의 겉넓이와 부피를 구할 수 있다"]
  L_A[A수준: 과정 설명/일반화]
  L_C[C수준: 과정을 이해하고 구함]
  L_E[E수준: 일부만 계산]
  AS1[예시평가: 원뿔 부피 구하기]
  GL1[지침: 복잡 계산 시 계산기 허용·과정 설명 요구]
  LEX1[용어사전: 겉넓이=표면적/surface area]
  STD2[9수02-09: 다항식 덧셈/뺄셈]
  STD3[9수02-10: (단항×다항)/(다항÷단항)]
  STD4[9수02-19: 인수분해]

  %% 구조적 관계
  PROG -->|BELONGS_TO| SUB
  COURSE -->|UNDER_PROGRAM| PROG
  COURSE -->|HAS_DOMAIN| DOM3
  CE1 -->|IN_DOMAIN| DOM3
  STD1 -->|IN_DOMAIN| DOM3
  STD1 -->|SPECIFIED_BY| L_A
  STD1 -->|SPECIFIED_BY| L_C
  STD1 -->|SPECIFIED_BY| L_E
  AS1 -->|FOR_STANDARD| STD1

  %% 추론 관계
  CE1 -. ALIGNS_WITH .-> STD1
  L_E -->|PROGRESSES_TO| L_C -->|PROGRESSES_TO| L_A
  DOM3 -. GUIDED_BY .-> GL1
  CE1 -. HAS_LEXICON .-> LEX1

  %% 위계/전제 예시(대수 위계)
  STD2 -. PREREQ .-> STD3 -. PREREQ .-> STD4
```

## 3) 예시 인스턴스 다이어그램(유리수 대소관계)

```mermaid
graph TD
  DOM1[수와 연산 (01)]
  CE2[내용요소: 유리수의 대소관계/절댓값]
  STD5[성취기준: 9수01-03<br/>"유리수의 대소관계를 이해하고 표현"]
  STD6[성취기준: 9수01-04<br/>"절댓값의 의미와 활용"]
  L_C[C수준: 기본 계산/비교]
  L_D[D수준: 단순 판단]
  LEX2[용어사전: 절댓값 ↔ | |]
  AS2[예시평가: 옳은 부등호 고르기]

  CE2 -->|IN_DOMAIN| DOM1
  STD5 -->|IN_DOMAIN| DOM1
  STD6 -->|IN_DOMAIN| DOM1

  CE2 -. ALIGNS_WITH .-> STD5
  CE2 -. ALIGNS_WITH .-> STD6
  STD5 -->|SPECIFIED_BY| L_C
  STD5 -->|SPECIFIED_BY| L_D
  AS2 -->|FOR_STANDARD| STD5
  CE2 -. HAS_LEXICON .-> LEX2
```
